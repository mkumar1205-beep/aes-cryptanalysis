# AES-128 Implementation + Cryptanalysis

This project implements AES-128 from scratch and explores its security through differential cryptanalysis.

## Features
> AES-128 encryption and decryption
> GF(2^8) finite field arithmetic
> Key expansion (key schedule)
> Differential cryptanalysis on reduced rounds

## Project Structure
> `src/` → core AES implementation
> `cryptanalysis/` → attack and analysis
> `tests/` → correctness testing

## How to Run
```bash
python main.py
